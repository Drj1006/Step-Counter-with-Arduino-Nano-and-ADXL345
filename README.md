**STEP COUNTER BASED ON ARDUINO NANO AND ADXL345**

**Project Overview**
This is a step counter built using an Arduino Nano and ADXL345. It detects motion and helps to count steps and displays the result on the 16*2 12C LCD.

**Status**
Ongoing improvement

**Components Used**
•	Arduino Nano
•	ADXL345
•	16*2 12C LCD Display
•	Button * 2
•	Wires
•	Vero board / Printed Circuit Board
•	Battery

**Wiring of ADXL345 – Arduino**
•	VCC – 3.3V
•	GND – GND
•	SDA – A4
•	SCL – A5

**Wiring of 12C LCD – Arduino**
•	VCC – 5V
•	GND – GND
•	SDA – A4
•	SCL – A5


**Wiring of Buttons**
•	Power Button – Pin 7, GND
•	Pause/Reset Button – Pin 8, GND

**Libraries Used**
•	Adafruit Unified Sensor
•	Adafruit BusIO
•	Adafruit ADXL345
•	LiquidCrystal 12C by Frank de Brabander
